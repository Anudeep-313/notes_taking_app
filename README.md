## Adding Note

``` yaml
node app.js add --title="{TITLE}" --body="{BODY TEXT}"
```
## Removing Note
``` yaml
node app.js remove --title="{TITLE}"
```

## Listing All Notes
``` yaml
node app.js list
```

## Read a Note
``` yaml
node app.js read --title="{TITLE}"
```
